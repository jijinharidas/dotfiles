# Nvim Configuration

* Install nvim
	* For arch
	```sudo pacman -S neovim```
	* For ubuntu/debian
	```sudo apt install neovim```
	
* Install [Plug for Nvim](https://github.com/junegunn/vim-plug)
* Copy contents
``` cp init.vim ~/.config/nvim/```
* Install Plugins
	* Open nvim and run ```:PlugInstall```
